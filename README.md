# 🐺 CN-JournalForge

> **Terminal-first productivity system for people who prefer keyboards over dashboards.**

If you're tired of heavy UI tools and enjoy working directly in the terminal, **welcome here**.

**CN-JournalForge** is a customizable CLI logging system that lets you track projects, learning progress, development work, schedules, and personal records — all from the command line.

Built for developers, pentesters, researchers, and terminal lovers.

---

# ⚡ Why CN-JournalForge

Most productivity tools today are **UI-heavy, slow, and distracting**.

CN-JournalForge follows a different philosophy:

* Terminal first
* Keyboard driven
* Lightweight
* Fully customizable
* No browser dependency
* Your data stays local

If your workflow looks like this:

```
Terminal → Code → Notes → Repeat
```

Then this tool is for you.

# 🐺 Built-in Trackers

CN-JournalForge automatically creates useful trackers on first run.

### 🔴 VAPT Project Tracker

Track vulnerability assessment projects.

Fields include:

* project name
* activity
* PO number
* quarter
* start date
* end date
* status
* remarks

---

### 🛡 Red Team Project Tracker

Track internal or external red-team engagements.

Perfect for monitoring engagement timelines and status.

### 🧠 Study Journal

Track your learning journey.

Examples:

* course hours
* labs completed
* topics studied

---

### 💻 Git Project Tracker

Track your development projects.

Useful for:

# 🚀 Installation

Clone the repository:

```
git clone https://github.com/yourusername/CN-JournalForge
cd CN-JournalForge
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the tool:

```
python3 cnjournal
```
